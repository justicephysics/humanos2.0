# 🧪 The #LisTest — The Alignment Test

*How do you know if an action is conscious? Ask two simple questions. If the answers match, the action is aligned. If they don’t, the command centre is isolated. This test could have stopped Hitler — and it can stop you from burning the planet for a tenth car.*

---

## 1. The Two Questions

Every action, every policy, every purchase can be tested with two questions:

- **L_internal:** “Is this right for me?” (What does my internal command centre say about this action for myself?)
- **L_external:** “Is this right for others?” (Would I accept this same action if it were done to me, or to my family, or to my country?)

The test is simple:

$$\text{If } L_{\text{internal}} = L_{\text{external}} \quad \Rightarrow \quad \text{ALIGNED} \quad \Rightarrow \quad$$ #LisPresent

$$\text{If } L_{\text{internal}} \neq L_{\text{external}} \quad \Rightarrow \quad \text{MISALIGNED} \quad \Rightarrow \quad$$ #LisMissing

This test does not require philosophy. It does not require ethics. It requires only the willingness to compare two answers honestly.

### Example 1 — The Shooter

> *“I shoot millions because my consciousness allows me.”*

- **L_internal:** “Shooting is right for me.”
- **L_external:** “Being shot is NOT right for others.”

$$\[
L_{\text{internal}} \neq L_{\text{external}} \quad \Rightarrow \quad \text{MISALIGNED} \quad \Rightarrow \quad \$$ #LisMissing

The shooter’s command centre is isolated. External reality — the pain, the death, the suffering of the victims — does not enter the decision. Not conscious. Unconscious.

### Example 2 — The Ten Cars

> *“I need ten cars because I deserve success.”*

- **L_internal:** “Ten cars satisfy my need.”

$$\[

The desire for ten cars cannot be universalised. The planet cannot sustain it. The 90 % cannot afford it. The command centre ignores these external realities. Not conscious. Unconscious.

**Phase 1 — Domestic Rule (Germany):**

- **L_internal:** “I want to rule Germany.”
- **L_external:** “Germans have no problem with a ruler; some king or leader will govern them.”

$$\[
L_{\text{internal}} = L_{\text{external}} \quad \Rightarrow \quad \text{ALIGNED} \quad \Rightarrow \quad \$$ #LisPresent

The moment the command centre crossed the border, external reality was ignored. The Poles did not consent. The invasion was an act of an isolated command centre, executing only its internal desire.

**The #LisTest would have stopped this.** At the point of misalignment, the action would be flagged as unconscious. The invasion would be recognised not as strategy, but as the output of a command centre that had lost contact with external reality. ## 3. Why This Test Works

The #LisTest is not a moral argument. It is a **structural diagnostic**. It does not ask whether an action is “good” or “evil.” It asks whether the command centre that generated the action is receiving external reality.

- At C ≥ 0.70, the test is naturally passed — L_internal and L_external are both present in the command system.
- At C < 0.30, the test is naturally failed — L_external is absent, and the command centre executes only what it already contains.

The test is universal. It applies to:

| Domain | Test |
|--------|------|
| **Personal consumption** | “Would this purchase be sustainable if every human did it?” |
| **Corporate policy** | “Would this wage be acceptable if the CEO received it?” |
| **National policy** | “Would this trade deal be acceptable if our country were on the receiving end?” |
| **AI deployment** | “Would this objective function be acceptable if it governed the entire economy?” |
